<h2>✨ Features</h2>
<ul>
  <li>Google-inspired homepage layout (logo, centered search, action buttons, footer links)</li>
  <li>Functional search form using <code>https://www.google.com/search?q=...</code></li>
  <li><strong>I'm Feeling Lucky</strong> action button wiring</li>
  <li>Top navigation with Gmail, Images, app-grid icon, and Sign In</li>
  <li>Responsive hamburger menu (using semantic <code>&lt;details&gt;</code>/<code>&lt;summary&gt;</code>)</li>
  <li>Mobile-friendly layout with adaptive spacing and footer behavior</li>
</ul>

<h2>🚀 Run Locally</h2>
<ol>
  <li>Open <code>index.html</code> in any modern browser.</li>
  <li>Type your query in the search bar.</li>
  <li>Click <strong>Google Search</strong> (or press Enter) to open real Google search results.</li>
</ol>

<h2>🎨 Design Notes</h2>
<ul>
  <li>Neutral palette and rounded controls for a familiar modern search experience</li>
  <li>Google-like colored wordmark styling via semantic spans</li>
  <li>Careful spacing and shadows for hover/focus depth</li>
  <li>Responsive breakpoints for compact navigation and footer readability</li>
</ul>

<h2>📱 Responsiveness</h2>
<p>
  On smaller screens, primary desktop actions collapse and a hamburger-driven quick menu appears,
  preserving usability and visual clarity.
</p>

<h2>🛠 Tech Stack</h2>
<p><strong>Pure HTML5 + CSS3</strong> (no JavaScript required).</p>
